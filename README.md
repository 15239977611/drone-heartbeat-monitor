# 无人机智能心跳监测与航线可视化系统基于 Streamlit 实现的无人机实时心跳监测、航线规划与障碍物识别可视化平台，支持卫星地图展示、坐标转换、实时心跳包模拟与异常掉线报警。## 功能介绍- 航线规划：设置无人机起飞点 A（校园内）与降落点 B（校外）- 高清卫星地图：基于 ArcGIS 卫星影像底图展示航线与障碍物- 坐标系自动转换：GCJ-02 与 WGS-84 坐标兼容- 障碍物随机生成：在航线区间自动生成模拟障碍物- 实时心跳监测：每秒模拟心跳包，3 秒未接收自动掉线报警- 数据可视化：心跳序号趋势图、历史数据记录与清空## 运行方式### 本地运行```bashpip install -r requirements.txtstreamlit run app.py
